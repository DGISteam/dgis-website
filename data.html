Notation!

We plan to public DGIS data through serveral data release (DR) including the pipeline reduced datacube (standard fits file) and the high-level datacube produced by GIST. The data reduction & analysis detail can be found in data description page. After the DGIS data overview paper completed, 1/3 DGIS data will be released in DR 1. To download these datacubes, please register an account through THIS LINK. Then log in to access and download DGIS data. If you are interested please send us a breif introduction of the project, including:

  • Title.
  • Background information.
  • Summary of planned analysis/methodology.
  • Expected outcomes.
  • Required data products and thether you are happy to contribute to their creation.

Please send these information to dgisteam.nju@gmail.com.

Customlized code and programs can be found in Github.
